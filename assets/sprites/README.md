# Sprite assets

Put the game sprite PNG files in this directory. The code expects these exact filenames:

## Hero

- `hero_high_hp.png`
- `hero_mid_hp.png`
- `hero_low_hp.png`
- `hero_attack.png`
- `hero_damaged.png`
- `hero_defeated.png`
- `hero_victorious.png`

## Mathematical Slime

- `slug_idle.png`
- `slug_attack.png`
- `slug_damage.png`
- `slug_defeated.png`
